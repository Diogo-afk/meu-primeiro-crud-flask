# 🚀 Meu Primeiro CRUD Flask (Full-Stack)

Este é um sistema de gerenciamento de usuários completo, desenvolvido para colocar em prática conceitos de Back-end, Front-end e Banco de Dados.

## 🌐 Acesse o Projeto
O site está rodando ao vivo aqui: [LINK_DO_SEU_RENDER_AQUI]

## 🛠️ Tecnologias Utilizadas
*   **Python / Flask**: Lógica do servidor e rotas.
*   **SQLite**: Banco de dados relacional para persistência.
*   **Bootstrap 5**: Interface moderna e responsiva (Mobile Friendly).
*   **Werkzeug**: Segurança com Hashing de senhas.
*   **Render**: Hospedagem em nuvem (Cloud).

## ⚙️ Como rodar localmente
1. Clone o repositório.
2. Crie um ambiente virtual: `python -m venv venv`.
3. Instale as dependências: `pip install -r requirements.txt`.
4. Execute: `python app.py`.
