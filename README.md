# Meu Primeiro CRUD com Flask 🚀

Este é um sistema de cadastro simples desenvolvido para aprender as bases do desenvolvimento Web.

## 🛠️ Tecnologias utilizadas:
* **Python** (Back-end)
* **Flask** (Framework Web)
* **HTML5 e CSS3** (Front-end)
* **CSV** (Banco de dados simples em arquivo)

## 📋 Funcionalidades:
* [x] Cadastro de usuários (Create)
* [x] Listagem de usuários (Read)
* [x] Edição de dados (Update)
* [x] Exclusão de registros (Delete)
* [x] Validação de nomes duplicados
